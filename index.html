Aircraft Leak Test System

SLM+

Static Leakrate Monitor Plus
The SLM+ measures a component's pneumatic leakage rate while under pressure using either the flow make-up or the pressure decay methods. This dual range monitor may be either wall or battery operated. Color touch screen-controlled operation with data logging.

SLM

Static Leakrate Monitor
The SLM is an economical unit used to measure a component's pneumatic leakage rate while under pressure using either the flow make-up or the pressure decay methods. This single range monitor may be either wall or battery operated. Color touch screen-controlled operation with data logging.

Hydraulic Leak Rate Systems

HTS-HP

Hydro Testing System, High Pressure
The HTS-HP is a high-pressure color touch screen operated data logging water leakage rate testing system. This portable system measures component and piping system leakage rates while they are maintained under a constant user selected test pressure.
Learn More

HTS-LP

Hydro Testing System, Low Pressure
The HTS-LP is a low-pressure color touch screen operated data logging water leakage rate testing system. This portable system measures component and piping system leakage rates while they are maintained under a constant user selected test pressure.

HSTS

Hydro Static Testing System
The Hydro Static Testing Systems are designed to be used on test volumes that are expected to be essentially leak-tight. These systems are all custom designed and built to specifications. Designed around the user to ensure reduced set-up times.
